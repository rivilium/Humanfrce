# Contributing

This is a shared public repository. Keep changes small enough to review, pull before starting work, and avoid committing local-only configuration.

## Suggested Flow

```powershell
git pull origin main
git status
```

Make your changes, then:

```powershell
git add .
git commit -m "Describe the change"
git push origin main
```

For larger changes, use a branch and pull request instead of pushing directly to `main`.
